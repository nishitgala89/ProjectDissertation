This section is a mini project for sentiment analysis of top 3 stocks in a MF selected. Below are the steps 

Step 1 - Get the ticker names for all top3 companies using TickerLookup.py 
Step 2- Map Company with Tickers using Company_Ticker_Map.py and save it in Company_Ticker_Map.json
Step 3- Run FetchingNewsData.py to scrape the latest news 
Step 4 - Notebook to show Sentiment Analysis (SentimentAnalysis.ipynb)
